# npuml

A Node wrapper around [PlantUML](https://plantuml.com/command-line)

## Pre-requisites

- [Java](https://www.java.com/en/download/)
- [Graphviz dot](https://plantuml.com/graphviz-dot)

## Usage

run with `npx`

```bash
npx npuml -h
```

Or install globally

```bash
npm install -g npuml
puml -h
```
